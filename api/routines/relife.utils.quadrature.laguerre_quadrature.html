laguerre_quadrature#

relife.utils.quadrature.laguerre_quadrature(func, a, deg=10)[source]#

Numerical integration of \(f(x) * exp(-x)\) over the interval \([a, \infty]\)

func must accept (deg,), (deg, n) or (deg, m, n) array shapes It must handle at least 3 dimensions. a can be zero with ndim <= 2.
